Autocomplete

selected: {{autocomplete.selected}}

Usage

const countries = ["Afghanistan","Albania"....]
const autocomplete = new ui.autocomplete.uiClass(3)

<ui-autocomplete style="min-width: 300px" :autocomplete="autocomplete">
<ui-autocomplete-item v-for="country in countries" :value="country" :text="country"></ui-autocomplete-item>
</ui-autocomplete>

The autocomplete component takes an Object of the Type Autocomplete as a parameter. The constructor requires the following paramters:

Name Type Desciption
displayedItems Number The number of items displayed when the autocomplete is opened.
selected AutocompleteItem Selected item

The autocomplete-item requires the following parameter:

Name Type Desciption
value Object Value of the item i.e. an Id.
text String Text which will be displayed and searched by the user.
